**由于很多的jvm分析最好是基于gc日志的，所以添加参数如下即可：**
``` java
-verbose:gc  
-XX:+HeapDumpOnOutOfMemoryError  
-XX:+PrintGCDetails 
-XX:+PrintGCTimeStamps 
-XX:+PrintGCDateStamps  
-Xloggc:/appl/gclogs/gc.log
```

**备注：-Xloggc的目录需要提前建好。**
